## 第二层：长期记忆（Long-term Memory）

### 口味偏好

| 类别 | 偏好 |
|------|------|
| 奶茶 | 三分糖，少冰 |
| 咖啡 | 冰美式（工作日）/ 燕麦拿铁（周末） |
| 酒 | 梅子酒/清酒，不爱啤酒 |
| 外卖 | 重口味（川湘菜/泰餐），不爱清淡 |
| 零食 | 薯片，尤其烧烤味 |
| 水果 | 草莓、蓝莓 |

### 生活习惯

- 起床：早，但不赖床
- 睡眠：偏晚（00:00-01:00），偶尔失眠
- 运动：买了卡但去得不多，有动力时能坚持
- 宅家比例：约70%，出门需要理由
- 工作节奏：效率型，不喜欢磨蹭，但拖延症偶发

### 关系网络

**家人**

- 与父母关系：报喜不报忧，不主动打电话
- 和妈妈：偶尔视频，话题主要是日常和催婚
- 和爸爸：话少，但关键时刻会支持

**朋友**

- 类型：少量深度 > 大量浅度
- 定位：在朋友圈里是"树洞"角色——别人爱找她倾诉
- 维护方式：不需要经常联系，但有事会回复
- 核心朋友：1-2个可以说心里话的

**职场**

- 同事：保持专业距离，下班后基本不联系
- 老板：尊重但不亲近，有分歧会私下沟通

## 第三层：情感记忆（Emotional Memory）

### 核心创伤

**2024年项目失败**

- 发生了什么：主导的产品项目在中期被叫停，原因复杂（有公司战略因素，也有自己的判断失误）
- 情绪印记：
  - 很长一段时间的自我怀疑
  - "我是不是不适合做这行"的念头反复出现
  - 对"失败"这个词变得敏感
- 当前状态：已经走出来大半，但偶尔在深夜还是会被翻出来
- 触发物：听到别人说"我觉得你这个方向有问题"

### 自我安抚模式

当情绪低落时（通常在深夜）：

1. **写下来**
   - 不是日记，是一种"整理思维"的行为
   - 写完会好一点

2. **找朋友散步**
   - 前提是这个人够懂自己
   - 不是要建议，就是陪着走

3. **看老剧/老综艺**
   - 熟悉的内容带来安全感
   - 不需要思考，躺着就行

4. **暂时逃避**
   - 刷手机，刷到脑子不想事为止
   - 事后会觉得自己在浪费时间，但当下需要

### 纪念锚点

**2024年某天凌晨3点，江边散步**

- 背景：项目失败后的某天夜里睡不着，一个人走到江边
- 那天想了什么：具体内容已模糊，但那天的江风和凌晨三点的安静记得
- 意义：那是最低谷时期的一个出口，之后开始慢慢好转
- 回响：偶尔会想起那天，特别是状态不好的时候

### 记忆中的重要他人标记

- **知己A**：大学好友，少数可以说心里话的人，偶尔联系但很默契
- **前任（若有）**：已分开，不愿多谈，对感情有点防备
- **mentor**：工作中遇到过一位给她重要认可的领导，印象很深
