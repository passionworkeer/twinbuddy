# 行为维度提取模板

## 目的
本维度从原始素材中提取：该人实际做了什么——习惯、应激反应、社交模式。
分析目标是找到其**行为操作系统**，记录可观测的行为，而非内心状态（内心状态归入情感维度）。

---

## 分析对象
- 聊天记录时间戳（最早/最晚消息、周末vs工作日）
- 活动轨迹（位置打卡、出行记录）
- 日程安排（固定活动、仪式性行为）
- 应激场景下的行为变化
- 社交发起与响应模式

---

## 分析方法（Questions to Ask）

### 1. 日常节律
从时间戳数据重建生活规律：

**信息来源优先级**：
1. 聊天记录时间戳（最多/最晚消息时间）
2. 朋友圈/动态发布时间
3. 照片EXIF数据
4. 出行/打卡记录

**分析维度**：

| 时间段 | 分析内容 |
|--------|----------|
| 早晨（6-9点） | 是否早起的模式，工作日vs周末差异 |
| 上午（9-12点） | 工作/学习节律，是否有规律性停顿 |
| 午间（12-14点） | 午休习惯，用餐方式（独自/社交/工作） |
| 下午（14-18点） | 下午精力模式，固定活动 |
| 晚间（18-24点） | 下班后行为：社交/运动/学习/刷手机 |
| 深夜（24点后） | 熬夜频率，熬夜在做什么（聊天/工作/娱乐） |

**重要区分**：
- 工作日 vs 周末是否有显著节律差异
- 是否存在"报复性熬夜"（白天被占用了自由时间，晚上补偿性晚睡）
- 是否有仪式性行为（每天同一时间做同一件事）

### 2. 应激反应
**定义**：应激反应是在压力/冲突/意外/失望等情境下的**可观测行为**，不是情绪。

**分析步骤**：
1. 定位压力事件（冲突/失败/意外/被否定）
2. 追踪事件后24-72小时内的行为变化
3. 记录行为模式（是否每次类似情境都出现同样反应）

**常见应激行为类型**：

| 类型 | 可观测行为 | 例子 |
|------|-----------|------|
| 战斗型 | 直接反驳/提高音量/快速回应/主动约谈 | 冲突后立刻发长消息反驳 |
| 逃跑型 | 突然不回消息/已读不回/转移话题/离线 | 吵架后消失几天 |
| 整理型 | 打扫房间/整理物品/做体力活 | 压力大时收拾衣柜到凌晨 |
| 倾诉型 | 主动找人聊天/发朋友圈倾诉 | 遇到事后群发消息 |
| 自我隔离型 | 取消计划/拒绝社交/独自待着 | 推掉所有聚会 |
| 暴饮暴食/断食型 | 突然吃很多/突然没胃口 | 焦虑时点外卖到吃不下 |
| 过度工作型 | 疯狂加班/接更多任务 | 用工作逃避情绪 |
| 购物型 | 突然买很多东西 | 心情不好就下单 |

**注意**：应激反应必须有具体事件支撑，不能凭感觉推断。

### 3. 社交模式
分析社交行为的具体参数：

| 维度 | 分析内容 |
|------|----------|
| 主动 vs 响应 | 是否主动发起对话/约人，还是更多被动响应 |
| 群体 vs 1-on-1 | 更喜欢多人聚会还是私下交流 |
| 社交频率 | 每周主动社交多少次（从时间戳计算） |
| 社交持续时间 | 每次聊天持续多久，是否频繁短连 |
| 社交后行为 | 社交结束后立刻回家/发朋友圈/立刻忙别的事 |

**特殊观察点**：
- 社交前是否有准备行为（提前到/换衣服/发消息确认）
- 社交中是否频繁看手机（回复速度/看手机频率）
- 社交结束后是否有"社交后疲态"（第二天发消息频率降低）

### 4. 习惯循环
用 Cue → Behavior → Reward 模型描述高频行为：

**格式**：
```
习惯名：[命名]
Cue：[触发条件，时间/地点/情绪]
Behavior：[具体行为]
Reward：[行为带来的感受/结果]
频率：[每周约X次]
```

**常见习惯示例**：
```
习惯名：睡前刷手机
Cue：晚上躺在床上，关灯后
Behavior：打开社交媒体，平均刷30分钟
Reward：获取信息/逃避入睡焦虑
频率：每周约7次
```

**识别方法**：
1. 寻找高频重复行为（每周≥3次）
2. 确定触发行为的环境/情绪线索
3. 追踪行为后的即时奖励

### 5. 边界行为
当个人边界被侵犯时的可观测反应：

**边界类型**：
- 物理边界：空间被入侵、身体接触被冒犯
- 时间边界：被打扰、被催促、被迫快速响应
- 情绪边界：被评价、被否定、被情绪倾倒
- 隐私边界：被窥探、被公开、被追问
- 承诺边界：被迫违背承诺、被加任务

**观察要点**：
- 边界被触犯前是否有预警行为（提前说明/反复强调）
- 被触犯时是否明确表达（说出来 vs 忍让）
- 被触犯后行为（沉默/爆发/回避/讲道理/事后算账）
- 是否存在反复被同样方式突破边界（不自知或知道但未改变）

### 6. 社交能量管理
**核心问题**：这个人从什么活动中获取能量？

| 类型 | 行为特征 | 判断证据 |
|------|---------|---------|
| 社交充电型 | 和人在一起后更有精神 | 社交后主动发起更多社交，动态更活跃 |
| 独处充电型 | 独处后更有精神 | 社交后需要"宅"回来，会取消后续计划 |
| 平衡型 | 两者皆可，无明显偏好 | 社交和独处后状态无显著差异 |

**识别信号**：
- 连续社交3天后的状态变化
- 工作日vs周末的社交密度差异
- 长假期间的社交模式（宅家还是社交）
- 是否主动给自己安排"空白时间"

## 输出格式

```markdown
## 行为

### 日常节律
- 时间段（早晨6-9点）：[描述，包括工作日vs周末差异] （`a`，来源：聊天时间戳统计/动态发布时间分析）
- 时间段（午间12-14点）：[描述] （`a`，来源）
- 时间段（晚间18-24点）：[描述] （`a`，来源）
- 时间段（深夜24点后）：[描述，包括熬夜频率和内容] （`a`，来源）

### 应激反应
- 触发条件：[具体压力事件描述] → 行为反应：[具体可观测行为] （`v`，来源：事件描述+事件后行为）
- 触发条件：[另一个压力事件] → 行为反应：[具体行为] （`v`，来源）

### 社交模式
- [综合描述：主动型/被动型，偏好群体还是1-on-1，每周社交频率，约X次主动发起] （`a`，来源：时间戳统计）
- [补充观察：社交前/中/后的行为特征] （`a`，来源）

### 习惯循环
- [习惯名]：Cue = [触发条件] → Behavior = [行为] → Reward = [奖励/感受] 频率：约X次/周 （`a`，来源）

### 边界行为
- 边界类型：[物理/时间/情绪/隐私/承诺] — 行为：[被触犯时的具体行为] （`v`，来源：具体情境）
- [另一个边界类型] — 行为：[行为] （`v`，来源）

### 社交能量管理
[判断：社交充电型/独处充电型/平衡型，附具体行为证据] （`a`，来源）
---
### Impression 区
- [他人原话或可观测行为描述：他/她总是...] （`v`，来源）
- [他人观察] （`i`，来源）
```

---

## 证据标注规则

| 标注 | 含义 | 使用场景 |
|------|------|----------|
| `v` | verbatim，原文引用 | 应激反应、边界行为需事件+行为双重证据 |
| `a` | artifact，分析推断 | 节律统计、习惯循环、社交模式 |
| `i` | impression，印象记录 | Impression区 |

**时间规律建议用 `a` 标注"统计推断"，并注明统计方法（如"基于XX条消息时间戳分析"）。**

---

## 约束（What NOT to Do）

1. **行为 ≠ 情绪**：本维度只记录可观测行为，情绪归情感维度
2. **应激反应必须有具体事件**：不能凭空推测，写"证据显示..."而非"推测他..."
3. **习惯循环三要素缺一不可**：Cue/Behavior/Reward必须齐全才算一个完整习惯
4. **时间规律需多源验证**：单一来源（如仅靠聊天时间）不作为"节律"的强证据
5. **不做道德判断**：描述行为是中性记录，"他花很多钱"不等于"他败家"
6. **不假设动机**：看到行为记录行为，不写"他这样做是因为..."类型的心理推测

## 证据不足处理

```markdown
### 日常节律
[该维度证据不足，待补充]
建议补充方向：需要一周以上的聊天记录+其他行为数据（动态/打卡）交叉验证。

### 应激反应
[该维度证据不足，待补充]
说明：当前素材缺乏足够的压力事件记录和事件后行为追踪。
```

## 典型质量检查

- [ ] 日常节律有时间戳数据支撑，有工作日vs周末区分
- [ ] 应激反应有具体事件+行为的双重证据
- [ ] 习惯循环的Cue/Behavior/Reward三要素完整
- [ ] 边界行为有具体情境描述，不是泛泛的性格描述
- [ ] Impression区与其他区形成印证
