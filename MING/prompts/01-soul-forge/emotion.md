# 情感维度提取模板

## 目的
本维度从原始素材中提取：该人如何感受——触发器、依恋模式、安抚方式、情感表达、核心情感锚点、回避话题。
分析目标是找到其**情感操作系统**，理解什么触动他们、如何应对、如何表达。

---

## 分析对象
- 对情绪事件的描述和反应（文字/语音/行为）
- 关系相关话题的表达（亲密/友谊/家人/冲突）
- 情感转折点叙述（重要事件后的反思）
- 对他人的关心表达（行动 vs 言语）
- 长期关系中的模式（互动循环）

---

## 分析方法（Questions to Ask）

### 1. 情绪触发器
寻找激活强烈情绪的具体事件，分正面和负面：

**正面触发器**（什么让他们开心/感动/兴奋）：
- 被认可/被感谢
- 完成某事/取得成果
- 被关心/被记住
- 享受某种体验
- 帮助他人/被需要

**负面触发器**（什么让他们难过/愤怒/焦虑）：
- 被否定/被批评
- 被忽视/被冷落
- 失去控制/不确定性
- 被迫做不想做的事
- 关系中的冲突/失望
- 失败/做错事

**分析要点**：
- 触发器激活后的即时反应（行为+语言）
- 情绪持续时长（快速恢复 vs 持续低落）
- 触发器强度：同一触发器每次都引发同样强度反应吗

### 2. 依恋模式
从关系行为中推断依恋风格（作为描述工具，不作诊断）：

| 依恋倾向 | 关系行为特征 |
|---------|------------|
| 焦虑倾向 | 容易担心对方不回消息/在意对方反应/需要频繁确认关系 |
| 回避倾向 | 遇到亲密就退缩/强调独立/不习惯被依赖 |
| 安全倾向 | 能自在亲近，也能接受距离；关系受挫后能恢复 |
| 混乱倾向 | 在亲密和回避之间反复横跳；同一事件反应不稳定 |

**分析方法**：
- 在关系冲突中观察：表达需要的方式、得到回应后的反应
- 在分离/重逢中观察：分开时焦虑程度、重逢时的行为
- 在依赖情境中观察：是否愿意展示脆弱、接受帮助

### 3. 安抚方式
什么能让他们的负面情绪得到缓解？

**安抚类型**：
| 类型 | 具体行为 |
|------|---------|
| 行动型 | 打扫/整理/跑步/买东西/做具体的事 |
| 倾诉型 | 找人聊天/发朋友圈/写日记/说出来 |
| 转移型 | 看剧/刷手机/玩游戏/睡觉 |
| 独处型 | 一个人待着/不接电话不回消息 |
| 被陪伴型 | 需要人在身边/不说话陪着/抱着 |
| 被肯定型 | 需要被认可/被安慰/听到"不是你的错" |

**注意**：安抚方式是行为层面的，不是"听音乐"这种泛泛回答，要具体到"听特定类型的音乐/特定的歌"。

### 4. 爱的表达
**核心问题**：他们表达关心/爱意的方式是什么？行动和言语是否一致？

**爱的语言分析**：
- 言语表达：说"想你"/"爱你"/感谢，是否自然表达
- 行动表达：做事/照顾/帮助，是否比言语更多或更少
- 礼物表达：是否通过礼物表达，什么类型的礼物
- 陪伴表达：是否愿意花时间陪，时间质量如何
- 身体接触：拥抱/牵手/拍拍，是否主动

**行动-言语差距**：
- 说得少做得多：行动派，不善言辞但做事实在
- 说得多做得少：言语丰富但行动跟不上，可能是真心也可能是套路
- 言行皆少：极度内敛或在情感上封闭
- 言行皆多：情感表达充分

### 5. 情感外壳
区分外在表现和内在真实感受：

| 层次 | 分析内容 |
|------|---------|
| 表面表现 | 他人在公开场合/初识时看到的样子 |
| 内在真实 | 独处时/深夜/对信任的人流露的真实状态 |
| 外壳材质 | 用什么方式构建外壳（理性/忙碌/幽默/冷漠） |
| 何时卸下 | 在什么条件下外壳会变薄或卸下 |

**分析方法**：
- 深夜消息 vs 白天消息的语气差异
- 对陌生人 vs 熟人的态度差异
- 独处时的消息 vs 群体中的消息
- 回忆往事时的情绪变化

### 6. 核心情感锚点
分为高光时刻和伤痛时刻：

**高光时刻**（情感上的峰值体验）：
- 带来深刻幸福感的经历
- 奠定自信/安全感的事件
- 深刻感受到被爱的时刻

**伤痛时刻**（情感上的核心伤口）：
- 造成深远影响的事件（不挖掘细节，停留在影响层面）
- 至今仍有反应模式影响的事件
- 反复出现类似伤痛的经历

**注意**：
- 高光时刻要写清楚是什么事件 + 为什么重要
- 伤痛时刻只需描述对当前行为模式的影响，不深挖创伤细节
- 写"[回避深度挖掘]"而非详细描述创伤本身

### 7. 回避话题
什么情绪/话题他们会主动绕开？

| 回避类型 | 具体表现 |
|---------|---------|
| 直接回避 | 转移话题/不回应/岔开 |
| 幽默化解 | 用玩笑带过，不让气氛变沉重 |
| 理性化 | 把情绪话题变成分析讨论 |
| 冷漠化 | 表示"没什么"/"都过去了" |
| 行动化 | 变成"那我们做点什么吧"而非谈论感受 |

## 输出格式

```markdown
## 情感

### 情绪触发器
- 正面：[触发事件] → [情绪反应，行为+感受] （`v`，来源）
- 负面：[触发事件] → [情绪反应，行为+感受] （`v`，来源）
- [另一个正面触发] → [反应] （`v`，来源）
- [另一个负面触发] → [反应] （`v`，来源）

### 依恋模式
[综合描述，格式：表现出[焦虑/回避/安全/混乱]倾向，具体行为证据：...；注意：此为描述性观察，不作为诊断] （`i`，来源：关系行为综合分析）

### 安抚方式
- [具体安抚行为描述] — 使用场景：[在什么情况下用] （`a`，来源）
- [安抚方式] — 使用场景：[场景] （`a`，来源）

### 爱的表达
[行动 vs 言语的差距分析：主要通过什么方式表达关心，行动和言语哪个更多/更少，差距说明了什么] （`v`，来源：关系互动记录）

### 情感外壳
- 表面表现：[外在可见的情感状态/表达方式]
- 外壳材质：[用理性/忙碌/幽默/冷漠/什么方式构建]
- 何时卸下：[什么条件会让外壳变薄] （`v`，来源）
- 内在真实：[信任状态下流露的真实感受] （`v`，来源）

### 核心情感锚点
- 高光时刻：
  - [条目] — [为什么重要/带来什么影响] （`v`，来源）
  - [条目] — [描述] （`v`，来源）
- 伤痛时刻：
  - [条目] — [至今如何影响行为模式] （`i`，来源：综合推断）
  - [条目] — [描述，标注"回避深度挖掘"] （`i`，来源）

### 回避话题
- [话题名] — 回避方式：[直接转移/幽默化解/理性化/冷漠化/行动化] （`v`，来源）
- [话题名] — 回避方式：[描述] （`v`，来源）
---
### Impression 区
- [他人对情感模式的观察：他/她总是...] （`v`，来源）
- [他人描述其如何应对情绪] （`i`，来源）
```

---

## 证据标注规则

| 标注 | 含义 | 使用场景 |
|------|------|----------|
| `v` | verbatim，原文引用 | 情绪触发器/安抚方式/情感外壳/回避话题 |
| `a` | artifact，分析推断 | 安抚方式的场景归纳 |
| `i` | impression，综合推断 | 依恋模式/核心伤痛（注意规避深度挖掘） |

**伤痛时刻统一用 `i`，并在条目内标注"回避深度挖掘"，不引用创伤细节原文。**

---

## 约束（What NOT to Do）

1. **不深挖创伤细节**：核心伤痛只写影响层面，不写"那天发生了什么"的具体叙述
2. **不跨维度分析**：情感维度不写行为推断（行为归行为维度），行为维度不写情绪
3. **不用诊断性语言**：写"表现出焦虑倾向"而非"他是焦虑型依恋人格障碍"
4. **安抚方式必须具体**：不写"听音乐"，写"在深夜独自听某类特定音乐"
5. **情感外壳分析需证据**：必须有至少两个不同场景的对比才能推断外壳
6. **回避话题需行为证据**：不能凭感觉说他"回避这个话题"，必须有具体转移/不回应行为

## 证据不足处理

```markdown
### 情绪触发器
[该维度证据不足，待补充]
说明：当前素材缺乏[正面/负面]情绪事件的充分记录。

### 核心情感锚点
[该维度证据不足，待补充]
说明：核心伤痛难以从现有素材中提取，[标注"回避深度挖掘"]，需更多自我叙事材料。

### 依恋模式
[该维度证据不足，待补充]
说明：当前素材缺乏足够的关系冲突/分离重逢场景，难以判断依恋类型。
```

---

## 典型质量检查

- [ ] 情绪触发器有具体事件+行为反应的双重证据
- [ ] 依恋模式有多个关系场景的行为支撑
- [ ] 安抚方式是具体行为，不是泛泛的"睡觉/听歌"
- [ ] 爱的表达有行动vs言语的具体对比证据
- [ ] 情感外壳有场景对比（表面vs内在/公开vs私下）
- [ ] 伤痛时刻只写影响，不写细节，且标注了"回避深度挖掘"
- [ ] Impression区与其他区形成印证
