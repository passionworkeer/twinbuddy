# PRD：平行宇宙叙事（视频命运修改器）

## 1. 产品概述与核心价值

**一句话定位：** 视频播完后通过 GALgame 风格的选择卡片，让用户进入"平行宇宙"并获得一张 AI 生成的命运结局海报，可一键分享至评论区。

**核心价值主张：**
- **情绪价值：** 每次选择带来"如果当初选了另一条路"的代入感与命运感
- **社交传播：** 海报天然适合分享，评论区晒图形成二次传播飞轮
- **技术差异化：** 端到端 AI 生成（故事 + 海报），而非预设素材拼接

---

## 2. 用户场景与痛点

| 痛点 | 描述 |
|------|------|
| 视频看完即走 | 用户情感投入但无处释放，缺乏后续互动钩子 |
| 评论互动同质化 | 现有评论区只有文字留言，缺乏视觉化表达 |
| AI生成结果雷同 | 现有AI工具生成内容千篇一律，用户缺乏"私人定制"感 |
| 分享门槛高 | 用户想分享但需要截图+配文，流程繁琐 |

### 4.3 Seed一致性机制

```
技术实现：session_id + 选择索引 → hash → seed
原理：同一用户对同一视频的同一选择，
     始终获得相同结局故事（seed固定）

session_id = f"{user_id}:{video_id}:{choice_index}"
seed = hash(session_id) % (2**32)

LLM调用时传入：
  "seed": seed  # 平台API支持seed参数
  "temperature": 0.75
```

### 4.2 关键设计点

- 同一视频同一用户多次生成结果一致（seed 机制）
- 添加 `video_theme` 和 `random_keyword` 作为强制约束变量
- `max_tokens: 300`，`temperature: 0.85`

| 故障 | 应对 |
