# PRD：场景DNA解码器

## 1. 产品概述与核心价值

**一句话定位：** 基于VLM的实时视频场景解读工具，通过"基因检测报告"风格的视觉卡片，将任意视频场景中隐藏的文化符号、阶层信号与审美基因翻译成用户可感知、可传播的情绪内容。

**一句话竞争力：** 把"懂，但说不出来"的生活洞察，变成一份人人都能转发分享的"社死诊断书"。

---

## 2. 用户场景与痛点

### 核心用户画像

| 用户类型 | 场景 | 痛点 |
|---------|------|------|
| 普通用户 | 刷到某个"很有感觉"的视频 | 感性认知无法结构化表达，缺少分享梗 |
| 内容创作者 | 需要给视频配文案 | 缺乏"场景洞察"的切入角度 |
| 品牌营销 | 分析消费者生活场景做洞察 | 需要快速理解场景背后的用户心理动机 |
| 黑客松评委 | 快速判断作品VLM能力深度 | 需要直观可感的Demo，而非技术汇报 |

### 三大核心痛点

1. **视觉信息过载，解读能力不足**：用户只能停留在"好看/无聊"的浅层评价
2. **洞察无法结构化输出**：即使感知到了"氛围感"，也缺乏将其转化为可分享内容的工具
3. **情绪价值缺少翻译器**：好的场景解读本身就是内容，但用户缺乏翻译能力

## 3. 产品功能详细设计

### 3.1 基因维度（精简版·4核心维度）

> ⚠️ **创新评审修正：** 原7维设计面面俱到但面面平庸，每个维度只有"检测到X → 结论Y"级别的浅分析，评委的期待是"比我看得更深"。精简为4个有洞察深度的核心维度，删除关系网络（最易流于表面）和审美谱系（与阶层基因高度重叠）。

| 维度 | 英文名 | 解读内容 | 示例输出 |
|------|--------|---------|---------|
| **阶层基因** | ClassDNA | 物品价位/品牌/摆放透露的社会经济位置 | "拼多多塑料袋×3 = 精打细算务实阶层" |
| **情绪状态** | EmotionCore | 环境凌乱度/装饰风格暗示的心理状态 | "外卖盒×咖啡杯 = 理想与现实的量子叠加态" |
| **时间戳信号** | TimeStamp | 年代元素/怀旧信号/时代背景 | "CRT显示器 + 搪瓷杯 = 90年代县城记忆编码" |
| **禁忌指数** | TabooIndex | 场景中不宜公开的私人痕迹（0-100）| "床底泡面箱×衣柜西装 = 职场身份的双重生活" |

> 💡 **安全模式设计：** 考虑到"禁忌指数"涉及私人场景评判，建议增加"安全模式/深度模式"开关。深度模式才输出禁忌指数，评委可选择进入，这本身就是一个Demo互动点。

### 3.3 设计规范

- **主色调**：深空灰背景 + 霓虹色（科技感）+ 琥珀色（温暖感）
- **字体**：无衬线粗体（标题）+ 等宽字体（数据/代码感）
- **装饰元素**：DNA双螺旋线条、基因序列ATCG字符作为背景纹理
- **核心hook**：顶部"综合判断标签"（如"都市打工人的精神乌托邦"）必须扎心有趣

## 4. 技术架构

### 4.1 系统架构

```
[用户上传/拍摄视频/图片]
    ↓
[帧提取器] → 关键帧序列（首帧+中间帧+尾帧，共3-5帧）
    ↓
[VLM模型] Qwen-VL-Max（主力）/ GPT-4o（降级）
    ↓
[符号学解析引擎] Prompt Engineering: 结构化输出JSON
    ↓
[多帧融合器] 3-5帧结果 → 投票/均值融合 → 统一DNA数据
    ↓
[叙事生成器] DNA数据 → 综合判断标签 + 场景叙事
    ↓
[卡片渲染引擎] HTML模板 → html2canvas → PNG/分享卡片
```

### 4.1.1 多帧融合方案

```
问题：3-5帧分别调用VLM后，结果如何融合？
方案：投票机制（多数表决）

对每个维度：
  - 各帧维度分数 → 取均值（四舍五入）
  - 各帧物品列表 → 取并集（去重），保留3-5个最有共识的物品
  - 各帧描述文本 → 取综合判断最尖锐的那个（而非平均）

综合判断标签：各帧分别生成1个标签 → AI从中选最有洞察力的1个
```

### 4.1.2 JSON Schema约束（Prompt必须强调）

> ⚠️ **技术评审修正：** 原Prompt没有强制要求JSON输出格式约束，会导致VLM输出不稳定。必须在Prompt末尾明确JSON Schema。

```python
scene_dna_prompt = """

【输出格式约束 - 必须严格遵守】
你必须且只能输出一个有效的JSON对象，不要输出任何解释性文字。
JSON必须符合以下Schema：

{
  "class_dna": {
    "score": 整数0-100,
    "items": ["物品1", "物品2", "物品3"],
    "description": "对该维度的一句话解读（20字以内）"
  },
  "emotion_core": {
    "score": 整数0-100,
    "items": ["物品1", "物品2"],
    "description": "20字以内解读"
  },
  "time_stamp": {
    "score": 整数0-100,
    "items": ["物品1", "物品2"],
    "description": "20字以内解读"
  },
  "taboo_index": {
    "score": 整数0-100,
    "items": ["物品1", "物品2"],
    "description": "20字以内解读"
  },
  "comprehensive_label": "综合判断标签（8字以内，必须扎心有趣）",
  "narrative": "场景叙事（80-120字，带洞察感，第二人称"
}

【综合判断标签生成Prompt（单独调用）】
从以下3个候选标签中，选择最有洞察力的1个：
候选标签：{three_label_candidates}

选择标准：
1. 是否足够扎心/有共鸣
2. 是否有反差感（表面和内在的反差）
3. 是否具备社交传播力（让人想截图/转发）

输出：直接输出被选中的标签，不要额外解释。
```

### 4.2 VLM选型策略

| 模型 | 优势 | 推荐指数 |
|------|------|---------|
| GPT-4o | 叙事能力强，指令遵循好 | ★★★★☆ |
| Qwen-VL-Max | 中文理解强，免费可用 | ★★★★★（黑客松首选）|
| Gemini 1.5 Pro | 长上下文，场景连贯性 | ★★★☆☆ |

**推荐：Qwen-VL-Max + GPT-4o降级方案**，优先使用国内可访问的Qwen-VL，演示现场切GPT-4o兜底。

### 4.3 符号学解析Prompt（核心资产）

```
你是一位文化符号学家和场景心理学家。请分析这张图片中的场景DNA。

分析维度（必须全部输出）：
1. 阶层基因：物品价位、品牌、摆放方式透露的阶层信息
2. 文化符号：亚文化标识物
3. 情绪状态：通过环境凌乱度、装饰风格判断心理状态
4. 审美谱系：装修风格溯源
5. 关系网络：人与空间、物品的关系暗示
6. 时间戳信号：年代元素、怀旧信号
7. 禁忌指数：场景中不宜公开的私人痕迹（0-100）

输出要求：
- 每个维度给出浓度分数（0-100）
- 每个维度给出3-5个检测到的具体物品/符号
- 最终给出一句"综合判断标签"（8字以内，扎心有趣）
- 输出一段场景叙事（80-120字，带洞察感）

语气：严肃分析 + 适度调侃，拒绝纯鸡汤，要有一针见血的锐度。
```

## 5. 随机关键词适配

### "马"的解码维度扩展

当关键词为"马"时，在7大基础维度上增加"马"关联解读层：

| 基础场景元素 | "马"关键词激活的新解读 |
|------------|----------------------|
| 马术俱乐部照片 | "马术 = 老钱阶层入场券，检测到圈层隔离信号" |
| 客厅里的马雕塑 | "装饰性马 = 财富稳态的象征，主人渴望稳定与掌控" |
| 办公室马形摆件 | "卷马 = 职场自我激励工具，奋斗逼的桌面图腾" |
| 草原骑马视频 | "马 = 逃离都市的田园想象，都市白领的精神出走" |

### 关键词适配触发机制

```
用户输入关键词"马"
    ↓
关键词匹配场景词库（马/马术/骑马/马蹄铁/马形摆件等）
    ↓
激活对应符号学解读模块
    ↓
原版7维报告 + 马主题解读 = 完整报告
```

**关键设计原则：** 用户感知不到关键词适配的痕迹，系统自动润物细无声地完成解读增强。
